GITHUB Y MENEJO DEL DOMINIO

Clase 08, 2023 Apartado de ALE

En esta clase conocimos sobre el Hosting y el dominio y el valor que cumple este en función a la página web. Este cumple la función de alojamiento web que ayuda dentro de la visualización masiva, sin el hosting las grandes páginas se caerian y dejarian en blanco su funcionamiento. Mientras que el dominio es el nombre que se le pone a nuestro sitio web.

COMPONENTES

  • WEBHOST: Es donde el sitio web está localizado fisicamente y se le denomina website.
  • DIRECCIÓN: Tambien conocido como URL
  • HOMEPAGE: Es la primera pagina que se abre cuando abrimos nuestro sitio web ¨Index¨
