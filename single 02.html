Sobreviviendo en programación

Clase 02, 2023 Apartado de ALE

Para esta clase profundizamos un poquito más en el lenguaje HTML para poder afinar y intervenir nuestros templates para el armado de nuestras bitacoras. En esta clase se indago sobre cada subtítulo y la función que cumple cada uno dentro de la cascada, también vimos un poco sobre el cambio de imágenes dentro del visual studio, ya para finalizar tocamos un poco el tema de atributos. Dentro de clase hicimos una demostración de como modificar el texto dentro de la carpeta index, visualizanado posteriormente en la pagina web. Hubo bastante interacción por parte de la clase.

PSDT: Cada clase estará siendo argumentada.
